# GameStore
Sample GameStore App

## Requirements for local debug

- dotnet core v3.1
- MySQL, Redis, Postgres
- docker

## Setup

### Init:
```
docker-compose up
```
Available on http://localhost/

### Clean up:
```
docker-compose stop && docker-compose rm -f
```
